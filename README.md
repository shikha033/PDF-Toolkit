## Installation

1. Clone this repository:
   ```sh
   git clone https://github.com/yourusername/pdf-toolkit.git
   cd pdf-toolkit
   ```

2. Install the required dependencies:
   ```sh
   pip install -r requirements.txt
   ```

## Usage

Run the application with:
```sh
python app.py
```

## Dependencies
- Python 3+
- Tkinter (built-in with Python)
- PyPDF2
